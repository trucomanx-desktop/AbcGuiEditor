# abc-gui-editor

A graphic user interface program to work with abc notation.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/abc_gui_editor/

```bash
pip install --upgrade abc_gui_editor
```

Using:

```bash
abc-gui-editor
```

## Install from source
Installing `abc-gui-editor` program

```bash
git clone https://github.com/trucomanx-desktop/AbcGuiEditor.git
cd AbcGuiEditor
pip install -r requirements.txt
cd src
python -m build
pip install dist/abc_gui_editor-*.tar.gz
```
Using:

```bash
abc-gui-editor
```

## Uninstall

```bash
pip uninstall abc_gui_editor
```
