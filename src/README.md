# abc-gui-editor

A graphic user interface program to work with abc notation.

![logo](https://raw.githubusercontent.com/trucomanx-desktop/AbcGuiEditor/main/screenshot.png)

## 1. Installing

To install the package from [PyPI](https://pypi.org/project/abc_gui_editor/), follow the instructions below:


```bash
pip install --upgrade abc_gui_editor
```

Execute `which abc-gui-editor` to see where it was installed, probably in `/home/USERNAME/.local/bin/abc-gui-editor`.

### Using

To start, use the command below:

```bash
abc-gui-editor
```
